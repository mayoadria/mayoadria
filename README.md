# 👨‍💻 Sobre mí

Recien graduado de **Desarrollo de Aplicaciones Multiplataforma (DAM)** con formación previa en **Sistemas Microinformáticos y Redes**, ambos cursados en el *Nicolau Copèrnic (Terrassa)*.

Mi perfil combina el **desarrollo de software (especialmente backend)** con un creciente interés en la **ciberseguridad y el hacking ético**, lo que me permite entender las aplicaciones no solo desde el punto de vista del desarrollo, sino también desde la perspectiva de su seguridad y posibles vulnerabilidades.

Me interesa construir sistemas eficientes, escalables y seguros, comprendiendo cómo interactúan las aplicaciones, las redes y la infraestructura que las soporta.

---

## 🧠 Enfoque técnico

- Desarrollo backend con Java y Spring Boot  
- Diseño y desarrollo de APIs REST  
- Programación orientada a objetos y buenas prácticas (SOLID)  
- Bases de datos relacionales y NoSQL  
- Redes y fundamentos de sistemas (TCP/IP, HTTP, DNS)  
- Ciberseguridad aplicada al desarrollo  
- Introducción al hacking ético y análisis de vulnerabilidades  

> “Entender cómo se construye un sistema es el primer paso para aprender a protegerlo.”
